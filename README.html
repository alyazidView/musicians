  • Target Variable: ‘Class’ such as Rock, Indie, Alt, Pop, Metal, HipHop, Alt_Music, Blues, Acoustic/Folk, Instrumental, Country, Bollywood,

  • Test dataset: 7,713 rows with 16 columns
